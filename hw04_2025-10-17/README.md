## Описание задания

Создать мобильное приложение, в котором:

- Создать классы животных (не менее 5–10), описать их свойства и методы.  
- Создать зоопарк, где животные могут **рождаться, умирать, есть, пить, гулять, играть, размножаться, болеть, выздоравливать**.  
- Процесс симуляции должен быть построен на **случайных числах**, а количество животных каждого вида также должно определяться случайным образом.

---

## Краткое описание решения

Реализовано на Dart/Flutter.
Приложение представляет собой консольную симуляцию жизни зоопарка, встроенную во Flutter-приложение.

- Созданы классы животных: **Lion**, **Elephant**, **Monkey**, **Zebra**, **Giraffe** — все наследуются от абстрактного класса `Animal`.  
- Каждый животный объект имеет состояния: `age`, `alive`, `sick` и методы поведения (`eat`, `drink`, `walk`, `play`, `getSick`, `recover`, `die`, `reproduce`).  
- Класс `Zoo` управляет коллекцией животных, выполняет случайные действия над ними и добавляет потомство.  
- Симуляция выполняется в виджете `ZooSimulation` с таймером, который каждые 2 секунды вызывает случайное действие и выводит результаты в интерфейс.

Основной код приложения в файле:
zoo/lib/main.dart

## Запуск проекта

Запустить приложение в режиме отладки:

   ```bash
   flutter run
   ```

   По умолчанию будет запущен симулятор зоопарка с текстовым логом на экране.

## Компиляция в релиз

Для сборки APK:

```bash
flutter build apk --release
```

Файл сборки появится по пути:

```

```

Для сборки под web:

```bash
flutter build web
```

Результат будет доступен в папке:

```
build/web/
```
