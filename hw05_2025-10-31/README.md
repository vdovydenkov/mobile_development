## Задание

Создать мобильное приложение, в котором отображается счётчик и две кнопки управления:

1. **Добавить** — увеличивает значение счётчика.  
2. **Сбросить** — обнуляет счётчик.  

Приложение должно показывать текущее значение счётчика и ограничение, при достижении которого появляется сообщение **«Лимит достигнут»**.  
При сбросе счётчика должно появляться сообщение **«Счётчик сброшен»**.

---

## Описание решения

Приложение реализовано на языке **Dart** с использованием **Flutter**.  
Главный экран (`MainScreen`) содержит отображение счётчика и две кнопки действий.

- При нажатии кнопки **Добавить** (`+`) значение счётчика увеличивается на 1.  
  Если счётчик достигает заданного лимита (в коде `_limit = 7`), отображается всплывающее сообщение (**SnackBar**) с текстом:  
  *«Лимит достигнут»*.

- При нажатии кнопки **Сбросить** (`↻`) значение счётчика обнуляется (`_counter = 0`) и отображается сообщение:  
  *«Счётчик сброшен»*.

Приложение использует компонент `Scaffold` с `AppBar`, `Column` для выравнивания элементов и `FloatingActionButton` для кнопок управления.

## Запуск и сборка приложения

### 1. Запуск в режиме разработки

Для запуска приложения на подключённом устройстве или эмуляторе выполните команду:

```bash
flutter run
````

После компиляции приложение откроется автоматически.

---

### 2. Сборка релизной версии

#### Android

Собрать APK-файл:

```bash
flutter build apk --release
```

Собранный файл будет доступен по пути:

```
build/app/outputs/flutter-apk/app-release.apk
```

#### iOS

Собрать релизную сборку:

```bash
flutter build ios --release
```

Собранный `.app`-пакет будет находиться в каталоге:

```
build/ios/iphoneos/
```

Для установки на устройство можно использовать Xcode или команду `flutter install`.
