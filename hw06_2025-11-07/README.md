## Задание

Создать мобильное приложение-игру:
- Пользователю отображается случайное число от 1 до 100 и слайдер во всю ширину экрана (альбомная ориентация).
- Пользователь касается слайдера в области, максимально близкой к заданному числу.
- Разница между фактическим числом и числом, которого коснулся пользователь — это результат игрового действия.
- Чем меньше разница, тем больше игровых баллов зарабатывает пользователь.
- После трех попыток-касаний выдается результат набранных баллов.

---

## Инструментарий

Приложение реализовано на языке **Dart** с использованием **Flutter**.  

---

## Запуск и сборка приложения

### 1. Запуск в режиме разработки

Для запуска приложения на подключённом устройстве или эмуляторе выполните команду:

```bash
flutter run
```

После компиляции приложение откроется автоматически.

---

### 2. Сборка релизной версии

#### Android

Собрать APK-файл:

```bash
flutter build apk --release
```

Собранный файл будет доступен по пути:

```
build/app/outputs/flutter-apk/app-release.apk
```

#### iOS

Собрать релизную сборку:

```bash
flutter build ios --release
```

Собранный `.app`-пакет будет находиться в каталоге:

```
build/ios/iphoneos/
```

Для установки на устройство можно использовать Xcode или команду `flutter install`.
